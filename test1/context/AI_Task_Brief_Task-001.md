# Task Brief: Task-001

## 目标
创建 20 个独立的 Python 脚本，分别输出对应编号的 “hello world 1” 至 “hello world 20”。

## 相关 Playbook
- [practice-001] 必须遵守: 暂无经验积累，完成任务后补充
- [anti-001] 必须避免: 暂无记录

## 实现步骤
1. 确定脚本命名规则（例如 `hello1.py` … `hello20.py`），保证编号与输出一致且无遗漏。
2. 为每个编号创建独立脚本，包含单行输出 `print("hello world N")`。
3. 快速自检（可抽样或逐个运行），确认每个脚本输出匹配编号。

## 验收标准
- [ ] 共 20 个独立 Python 脚本，编号连续 1-20，无缺失或重复。
- [ ] 每个脚本执行时仅输出对应的 “hello world N”，无额外内容或依赖。
