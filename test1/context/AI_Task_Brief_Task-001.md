# Task Brief: Task-001

## 目标
创建20个以编号命名的Python脚本，分别输出对应编号的 hello world 文本。

## 相关 Playbook
- [practice-001] 必须遵守: 待整理
- [anti-001] 必须避免: 待整理

## 实现步骤
1. 确定编号范围1~20，使用统一命名规则创建对应的Python脚本文件。
2. 在每个脚本中实现 `print("hello world <编号>")`，确保输出编号与文件名一致。
3. 逐个或批量运行脚本，确认输出格式正确且无缺漏。

## 验收标准
- [ ] 存在1~20共20个以编号命名的Python脚本，无缺失或额外文件。
- [ ] 每个脚本执行时输出的文本为 `hello world <编号>`，且编号与文件名匹配。
